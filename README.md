# AI-for-Math-calendar-2026
Calendar of 2026 deadlines related to AI for Math. Includes deadlines for 2027 conferences. Includes some theoretical computer science conferences related to logic and proof theory (my personal interest).

_Last verified: 2026-03-22_

## Gaps / to check later

- **IJCAR 2026:** exact final paper deadline should be re-checked on the live CFP before planning a submission; public pages/snippets are slightly inconsistent.
- **AI for Math Fund 2026 grant:** the public page gives the **abstract deadline** and later milestones, but not a precise public date for invited full proposals.
- **FSCD 2027 / CADE-31 (2027):** announced, but no public submission deadline yet.
- **Proof Theory school/workshop 2026 (Aussois), FoPSS 2026,, Rocqshop 2026, Proof Complexity 2026:** public event visibility exists, but no stable CFP/application deadline was found.
- **COLM 2026:** conference site / OpenReview timezone presentation may differ by one day around the full-paper deadline.
